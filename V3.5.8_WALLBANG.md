# Beep v3.5.8 - RAGEBOT WALLBANG ✅

## 🎯 NUEVA FEATURE: WALLBANG CON HEIGHT OFFSET

Sistema de wallbang agregado al ragebot para disparar sobre paredes y obstáculos.

---

## 🔧 CÓMO FUNCIONA

### Mecánica:
Agrega un offset vertical (Y) a la posición del target antes de apuntar:

```lua
-- RIVALS WALLBANG: Add vertical offset to shoot over walls
if Config.Combat.RagebotWallbang then
    aimPos = aimPos + Vector3.new(0, Config.Combat.RagebotWallbangOffset, 0)
end
```

**Resultado**: Disparas +Y studs arriba del enemigo, pasando por encima de paredes bajas.

---

## 🎮 CÓMO USAR

### 1. Activa el Ragebot
- Abre Beep (INSERT)
- Ve a **Combat** tab
- Activa **"Ragebot"**

### 2. Configura Wallbang
- Activa **"Ragebot Wallbang (+Y offset)"**
- Ajusta **"Ragebot Wallbang Offset"** slider:
  - **1-5 studs**: Paredes MUY bajas, cobertura pequeña
  - **5-8 studs**: IDEAL para RIVALS (default: 5)
  - **10-15 studs**: Paredes altas, edificios pequeños
  - **15-20 studs**: Edificios grandes, obstáculos muy altos

### 3. Recomendaciones
- Combinar con **"Ragebot Face Target"** para mejor precisión
- Usar con **"Ragebot Full Map"** para apuntar a través de todo el mapa
- Activar **"Ragebot NoClip"** para moverte a través de paredes

---

## 📊 CASOS DE USO

### ✅ FUNCIONA BIEN EN:
- **RIVALS**: Paredes de cobertura baja/media
- **Arsenal**: Cajas, barriles, obstáculos pequeños
- **Phantom Forces**: Cobertura baja, ventanas
- Cualquier juego con obstáculos de altura baja-media

### ⚠️ LIMITACIONES:
- **NO funciona con techos**: Solo offset vertical, no atraviesa objetos sólidos
- **Anti-cheat**: Muy detectable en juegos con validación server-side
- **Precisión**: Puede fallar si el offset es muy alto (dispara sobre la cabeza)
- **Física de balas**: Depende de cómo el juego maneja las balas

## 🐛 TROUBLESHOOTING

### "No le pego aunque apunto arriba"
- **Solución**: Reduce el offset (prueba 3-5 studs)
- **Causa**: Offset muy alto, disparas SOBRE la hitbox

### "Sigo chocando con paredes"
- **Solución**: Activa "Ragebot NoClip"
- **Causa**: NoClip desactivado

### "Me detectan muy rápido"
- **Solución**: Desactiva "Ragebot Auto TP/Fly" y "Ragebot Full Map"
- **Causa**: Muy agresivo = más detectable

### "Solo funciona a veces"
- **Solución**: Usa "Ragebot Face Target" + offset bajo (5 studs)
- **Causa**: Física de balas del juego o ángulo incorrecto

## 📝 NOTAS TÉCNICAS

### Orden de Aplicación:
1. Obtener posición del target (`target.Position`)
2. Aplicar **Wallbang offset** (`+ Vector3.new(0, offset, 0)`)
3. Aplicar **Prediction** (si está activo)
4. Snap camera a `aimPos`
5. Disparar (si Auto Shoot activo)

- ✅ Funciona con Auto TP/Fly
- ✅ Funciona con Prediction
