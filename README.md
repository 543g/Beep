## 🚀 Quick Start

**Installation:**
```lua
loadstring(game:HttpGet("https://raw.githubusercontent.com/765g/Beep/main/Beep.lua?t=" .. tostring(tick())))()
```

**First Launch:**
1. Execute the script above in your executor
2. Wait for the animated loading screen
3. Press `INSERT` to open the interface
4. Configure your preferred settings in each tab

## ✨ Features Overview

### 🎯 Targeting Systems
- **Assisted Aiming** - Configurable FOV radius, smoothness, and body part selection
- **Target Locking** - Lock onto specific targets with keybind (default: Q)
- **Auto-Action** - Automatic actions when conditions are met
- **Rapid Mode** - Enhanced action speed (up to 50 actions/sec)
- **Smart Switching** - Auto-switch to next target when current is eliminated
- **Range System** - Area-based automatic targeting with distance control
- **Advanced Modes** - Multiple targeting strategies (Closest, Health-based, Crosshair)
- **Game Profiles** - Optimized presets for popular games (Auto-detect available)

### 👁️ Visual Enhancements
- **Player Information** - Display names, distances, and identifiers
- **3D Highlighting** - See players through obstacles
- **2D Overlays** - Frame-based player marking
- **Skeletal Display** - Anatomical wireframe visualization
- **Connection Lines** - Tracers from viewpoint to players
- **Status Indicators** - Health and state visualization
- **Focus Markers** - Targeting reticle on specific body parts
- **Custom Colors** - Configurable color schemes for all visual elements

### ⚡ Movement & Physics
- **Speed Enhancement** - Adjustable movement speed multiplier with toggle key
- **Jump Boost** - Configurable jump power (50-300 range)
- **Flight System** - Full 3D movement with WASD + Space/Shift controls
- **Collision Toggle** - Phase through solid objects (F2 to toggle)
- **Quick Travel** - Ctrl+Click teleportation to cursor location
- **Infinite Jumps** - Remove jump limit for continuous air movement

### 🔧 Utilities & Customization
- **Brightness Control** - Enhanced visibility in dark environments
- **Atmosphere Clear** - Remove fog and weather effects
- **View Angle Adjustment** - FOV slider (70-120 degrees)
- **Connection Protection** - Anti-idle system
- **Status Display** - Draggable FPS and ping watermark
- **Theme Customization** - 6 built-in accent colors plus custom picker
- **Player Navigation** - Quick teleport to any player by name
- **Activity List** - Real-time display of active features with keybinds
- **Universal Search** - Find any setting across all tabs instantly
- **Safe Exit** - Clean shutdown with no traces

## ⌨️ Default Keybinds

| Key | Function |
|-----|----------|
| `INSERT` | Toggle main interface |
| `Q` | Lock/unlock current target |
| `E` | Toggle flight mode (when enabled) |
| `F2` | Toggle collision phase (when enabled) |
| `Left Ctrl` | Speed boost / Click teleport modifier |
| `Right Mouse` | Hold-to-aim mode (when configured) |
| `END` | Emergency stop (disable all features) |

*All keybinds are fully customizable in the interface*
